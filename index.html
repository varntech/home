Product Name

Smart Ball

"Smart Ball" is a very smart robot that has the autonomous movement capability and can react on dogs' behaviors and can optimize itself by analyzing the play pattern of dog. It moves autonomouly, no need to control. It has many actions and maneuver patterns, so it can attract dogs' interest. In addition, we will soon introduce the new kind of ball for cats.

  • Type : Robot
  • Category: IoT, Pet-Tech

Product Name

Smart Feeder

"Smart Feeder" is an automatic pet feeder that dispenses food on a schedule and a action. It is the hub for Smart Pet Home. It can set automatic mealtimes and your pet’s portion size based on activity monitoring results. So it will improve the quality of life for your friends.

  • Type : Feeder
  • Category: Iot, Pet-Tech

  • Type: STEAM Kit
  • Category: IoT, Tangiable Coding
